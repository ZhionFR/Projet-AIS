### Installations

Installation de libvirt, kvm et virt-manager :

```bash
$ sudo apt install qemu-kvm libvirt-daemon libvirt-daemon-system libvirt-clients libvirt-daemon-driver-qemu  

$ sudo apt install virt-manager  

$ sudo systemctl start libvirtd  

$ sudo systemctl enable libvirtd
```

### App web

Buildée par Claude code

ip actuelle : http://127.0.0.1:5000/
##### Stack technique :

| Élément         | Choix                                     |
| --------------- | ----------------------------------------- |
| Langage         | Python 3.13                               |
| Framework web   | Flask 3                                   |
| ORM             | peewee                                    |
| Base de données | SQLite (`messages.db`, fichier local)     |
| Templates       | Jinja2 (fourni avec Flask)                |
| Environnement   | venv (`requirements.txt` : Flask, peewee) |
```
App AIS/
├── app.py            # routes Flask + gestion connexion BDD
├── models.py         # ORM peewee (modèle Message) + SQLite
├── templates/
│   └── index.html    # page web (formulaire + liste des messages)
├── requirements.txt  # dépendances
└── messages.db       # base SQLite (générée)
```
